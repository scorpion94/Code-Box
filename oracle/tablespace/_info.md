## Available Commands

- [find_biggest_objects.sql](find_biggest_objects.sql) → Find biggest objects in Tablespace
- [get_path_from_datafiles.sql](get_path_from_datafiles.sql) → Get Path for Datafiles
- [get_tablespace_usage.sql](get_tablespace_usage.sql) → Get Tablespaces usage - Ignores free extends
- [get_tempfile_usage.sql](get_tempfile_usage.sql) → Get Size of Tempfile
